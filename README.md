# Honeycomb Oasis 3X Website

GitHub Pages-ready website for the Honeycomb Oasis 3X Rust server.

## Files

- `index.html` — main page
- `css/style.css` — design and responsive layout
- `js/timer.js` — live Eastern Time wipe-window countdown
- `images/poster.webp` — server poster
- `images/logo-192.png` and `images/logo-512.png` — site/app icons
- `favicon.ico` — browser tab icon
- `site.webmanifest` — installable app information
- `404.html` — page-not-found screen
- `robots.txt` and `sitemap.xml` — search-engine files
- `.nojekyll` — keeps GitHub Pages from processing the site with Jekyll

## Publish with GitHub Pages

1. Upload every file and folder in this package to the root of your GitHub repository.
2. Open repository **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose **main** and **/(root)**, then save.
5. After GitHub gives you the live address, replace the placeholder URL inside:
   - `robots.txt`
   - `sitemap.xml`
